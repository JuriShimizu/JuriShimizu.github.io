Vagrantで仮想環境を作成

これから勉強するために仮想環境をつくります。

仮想環境を作ってみる

Vagrant と VirtualBox を使って、仮想環境を作ってみます。

Vagrant をインストール

Download Vagrant – Vagrant

インストール後、バージョン確認

まずはboxを確認

Vagrantfileを作成

ネットワーク設定をする

vagrant のリロード
